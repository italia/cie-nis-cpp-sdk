Detailed Description

Main facilities.

This file contains the most important core functions of the entire NIS sdk. Just include nis.h in your source code to use this API.

int NIS_ReaderList ( char **  readers,
size_t *  len 
)

Obtain the connected reader list.

Parameters
[out]readersif the value len points to is 0 then readers will be automatically allocated (and automatically freed on deinit, so no user intervention is required). If not, readers must point to a pointer to a valid array allocated by the caller who is the owner of that array
[in,out]lencould point to a value of 0 for auto-allocation, or to the size of readers array when it's provided by the user
Returns
0 on success, negative on error. If negative, the abs value stands for the number of backends for which it was not possible to get the reader list

Read the NIS from the specified token.

Parameters
[in]handlethe handler of the reader previously obtained calling NIS_GetHandle()
[out]nisDataarray in which to store the NIS read back from the token. Should be long enough to contains the entire nis as a C-style sring, i.e. 12 (NIS) + 1 (null terminator) = 13 chars
[in]callbackif NULL the call is blocking and the NIS is copied inside nisData upon return, otherwise the function spawns a background thread and returns immediately. The thread will invoke the callback funcion passing to it the read NIS
[in]intervaltime in ms between polls
[out]theUID associated to the newly created context of execution
Returns
0 on success, negative on error
