Detailed Description

Main facilities.

This file contains the most important core functions of the entire NIS sdk. Just include nis.h in your source code to use this API.

int NIS_ReaderList ( char **  readers,
size_t *  len 
)

Obtain the connected reader list.

Parameters
[out]readersif the value len points to is 0 then readers will be automatically allocated (and automatically freed on deinit, so no user intervention is required). If not, readers must point to a pointer to a valid array allocated by the caller who is the owner of that array
[in,out]lencould point to a value of 0 for auto-allocation, or to the size of readers array when it's provided by the user
Returns
0 on success, negative on error. If negative, the abs value stands for the number of backends for which it was not possible to get the reader list
